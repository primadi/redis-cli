## A Simaple redis cli - Rclient

### Introduction

A very very simple redis-cli written by golang

### Usages

```shell
git clone git@github.com:liangwt/redis-cli.git
cd redis-cli/
go run main/main.go or go run main/main.go -h 127.0.0.1 -p 6379
```
